EveVision

The EVE Online UI you've always wanted.

If you play EVE, you are probably alt+tabbing to web-based tools every single time you play. These tools are one of the many things that makes EVE Online different. However, it also ruins the immersion of the beautiful world CCP built.

The era of having to leave EVE to play EVE is over.

EveVision is a EULA-legal way to extend the UI of EVE Online, bringing the tools you love directly into the game client.

Instructions

We've made it as simple as possible to use EveVision.

Just download and run the portable EXE from GitHub and log into EVE. No installation is required.
