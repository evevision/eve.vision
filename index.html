EveVision

The era of having to leave EVE to play EVE is over.

If you live in New Eden, you are probably used to having to alt+tab to web-based tools every single time you play. These tools are part of what makes EVE Online unique. However, having to do this also ruins the immersion of the beautiful world CCP built.

EveVision is a EULA-legal way to extend the UI of EVE Online, bringing the tools you love directly into the game client.

Instructions

We've made it as simple as possible to use EveVision. Just download and run the portable EXE from GitHub and log into EVE.

No installation is required. For more information, check out the readme.

If you run into any problems, please drop by the Discord for help!
